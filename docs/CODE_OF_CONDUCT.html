Code of Conduct

Our Commitment

We are committed to providing a welcoming, safe, and respectful environment for everyone, regardless of age, body size, disability, ethnicity, gender identity/expression, level of experience, nationality, personal appearance, race, religion, or sexual identity/orientation.

Expected Behavior

  • Be respectful and empathetic.
  • Give and receive constructive feedback.
  • Assume good intent and ask clarifying questions.
  • Focus on what is best for the project and community.
  • Help create an inclusive environment.

Unacceptable Behavior

  • Harassment, intimidation, or discrimination.
  • Trolling, insulting, or derogatory comments.
  • Personal or political attacks.
  • Publishing others’ private information without permission.
  • Any conduct that is inappropriate in a professional setting.

Scope

This Code of Conduct applies in all project spaces, including:

  • GitHub issues, PRs, and discussions
  • Project chat/community channels
  • Official events and communications

Enforcement

Maintainers are responsible for clarifying and enforcing standards of acceptable behavior and may take appropriate corrective action for violations.

Possible responses include:

  • Warning
  • Temporary suspension from participation
  • Permanent ban from project spaces

Reporting

If you experience or witness unacceptable behavior, report it to the maintainers via the repository’s private contact/security channel.

When reporting, include:

  • What happened
  • Where and when
  • Any supporting context (links/screenshots)

Reports will be reviewed promptly and handled with care.

Attribution

This Code of Conduct is inspired by the Contributor Covenant (v2.x) principles and adapted for this project.
